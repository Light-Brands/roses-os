## 4. VISUAL & BRAND GUIDELINES

### Color System

- **Primary background:** Neutral, warm tone (peachy neutral / rose clay)
  - Warm, inclusive, non-gendered
  - Consistent across all non-chakra slides
- **Chakra sections:**
  - Each slide uses the correct color for its chakra
  - Consistent, accurate chakra colors
  - Harmonized saturation so slides blend together
- **Gold accents:**
  - Used for headings, dividers, key phrases
  - Represents highest vibration, coherence, clarity

### Typography (open to suggestions)

- **Primary font:** Clean, modern serif or humanist sans-serif (e.g., Cormorant Garamond)
  - Elegant, spiritual, readable
- **Secondary font:** Simple sans-serif for body text (e.g., Inter)
- **Consistent hierarchy:**
  - Titles
  - Subtitles
  - Body
  - Callouts

### Visual Style

- Soft gradients
- Subtle particle or light textures
- Calm, grounded, sacred-tech feel
- No harsh contrasts
- No overly busy visuals
- Texture language: clay, linen, stone, paper, soft fogged light

### Consistency Rules

- Same background style across all non-chakra slides
- Same layout logic for each chakra section
- Same color logic throughout
- Same iconography style
- Gold used intentionally — never decoratively
