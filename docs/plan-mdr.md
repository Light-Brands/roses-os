# ROSE MEDITATION

## Teacher Visual Aid Manual — Plan & Brand Guidelines

Levels 1, 2 & 3

> Having a neutral color background that is unisex and warm. Except the chakra slides which can have the correct color for the chakra.

---

## 1. PURPOSE OF THIS MANUAL

This visual aid manual is designed to support teachers and facilitators while guiding students through the Rose Meditation – Levels 1, 2, and 3.

- It is **not** a student workbook
- It is **not** the full advanced aura-reading initiation
- It **is** a guided visual reference to support live teaching, embodiment, and energetic understanding

The manual helps teachers:

- Explain energetic concepts clearly
- Reference diagrams while teaching
- Maintain consistency across facilitators
- Reduce cognitive load during live sessions

## 4. VISUAL & BRAND GUIDELINES

### Color System

- **Primary background:** Peachy neutral or your suggestion
  - Warm, inclusive, non-gendered
- **Chakra sections:**
  - Consistent, accurate chakra colors
  - Harmonized saturation so slides blend together
- **Gold accents:**
  - Used for headings, dividers, key phrases
  - Represents highest vibration, coherence, clarity

### Typography (open to suggestions)

- **Primary font:** Clean, modern serif or humanist sans-serif
  - Elegant, spiritual, readable
- **Secondary font:** Simple sans-serif for body text
- **Consistent hierarchy:**
  - Titles
  - Subtitles
  - Body
  - Callouts

### Visual Style

- Soft gradients
- Subtle particle or light textures
- Calm, grounded, sacred-tech feel
- No harsh contrasts
- No overly busy visuals
